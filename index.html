Data Science Pay Trends & Predictions

An analytical exploration of salary trends in the field of Data Science.

Project Overview

This project analyzes the evolving salary trends in Data Science using datasets spanning the past decade. Insights were drawn using SQL, Python, and data visualization techniques to uncover meaningful patterns and future predictions.

Methodology

  • SQL: Data extraction and aggregation from databases.
  • Python: Data cleaning, trend analysis, and forecasting.
  • Data Visualization: Charts and interactive graphics for clear analysis.
