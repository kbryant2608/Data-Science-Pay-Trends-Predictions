# Data Science Pay Trends & Predictions  
Exploratory analysis and predictive modeling of global Data Science salaries from 2020–2023. This project analyzes how experience level, job role, and geographical location impact salary expectations in the Data Science field.

---

## 🔍 Project Objectives
1. Evaluate how experience level affects salary in different job roles.
2. Analyze salary predictability based on employee rank and experience.
3. Identify trends across regions and job types.

---

## 🛠️ Technologies Used
- **Python** (Pandas, NumPy)
- **Data Visualization**: Matplotlib, Seaborn
- **Jupyter Notebook**
- **CSV Data Management**
- **GitHub Pages** (for documentation)

---

## 🚀 Key Findings
- Senior-level positions tend to earn 20–30% more than junior roles across all regions.
- Data Engineers saw the highest growth in salary predictability based on experience.
- Western Europe has more predictable salary bands compared to North America.
